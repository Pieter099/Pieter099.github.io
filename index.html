About Me

My journey into tech has been a rewarding challenge, shaped by my determination to overcome obstacles and make a difference. My projects

Company Home Page
This project involves designing a responsive homepage for a fictional scuba gear company, "Deep Blue Explorer". Using Flexbox and Flexbox Media Query for the layout of the homepage. The goal is to create a visually appealing and user friendly homepage that effectively showcases the products that the company has to offer as well as the company team. Features included in this homepage are mainly a navigation bar the company mission statement a product gallery to showcase all the products of the company and a staff section for the company team, technologies like CSS Flexbox and Flexbox Media Query to ensure the layout adapts to different screen sizes. Technologies that is used in this project include HTML, CSS, Flexbox and media query.

Local club Home Page
This project involves designing a responsive home page for a fictional local club, Amateur model rockets using flexbox and responsive technologies for the layout. The goal is to create a visually appealing and user-friendly home page. A home page that efficiently showcases the local club's events and other additional information. Features of this home page include a navigation bar, a mission statement of the local club, a list of events for coming events during a particular month, a calendar of that particular month to showcase the days and dates the events take place, a photo gallery showcasing photos taken during the event days, as well as a contacts list where one can contact the club via email or telephone number or visit their social media platforms. This home page also consists of a light and dark mode, depending on which you would prefer, as well as responsive technologies, for example Flexbox, to adjust the home page of this local club according to the screen size of your mobile device. The technologies that are used within this project is HTML, CSS, and the media query technologies within CSS, as well as JavaScript.

Musical Home Page
This project involves designing a responsive home page for a fictional musical website, PlayVibes using CSS Flexbox, CSS Transition and Animation as well as other responsive technologies The goal and purpose of this web page is to create a visually appealing and user- friendly home page that efficiently showcases the videos shared from different people around the world of how they play their musical instruments This web page is also inspired by the media platform YouTube where one can share your content to the rest of the world. Features of this home page include a navigation bar on the left side of the screen a mission statement of this web page a list of videos that is categorically ordered according to the instruments a sign-up page and a contact list to contact the owner behind this web page via e-mail or view any updates on social media. Technologies that is used for this web page include HTML, CSS Flexbox, CSS Animation and Transition as well as CSS media query technologies.
